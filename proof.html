Unit Disc Covering Proof

Can any ten plane points be covered by non-overlapping unit discs?

Answer

Yes. Throughout this page, unit discs are closed, and "non-overlapping" means pairwise interior-disjoint; boundary tangency is allowed. Given any 10 points in the plane, some finite subcollection of a translated hexagonal packing of unit discs covers all 10 points.

If an alternate rule forbade even tangency, scale the triangular lattice so neighboring centers are 2 + epsilon apart. The density becomes 2 pi / (sqrt(3)(2 + epsilon)^2), which is still greater than 0.9 whenever, for example, 0 < epsilon < 0.0076. The same expectation proof then gives pairwise disjoint closed unit discs of radius 1.

The second question has a different answer than a search for a 10-point counterexample suggests: 10 is not a failure case. For the general problem, the exact minimum number of points that can force failure is an open discrete-geometry problem. Let that minimum be k. The hand-checkable published range is 13 ≤ k ≤ 50. The best reported computer-assisted upper bound improves this to k ≤ 45, but the cited 2012 paper marks that endpoint as coming from exhaustive computer search rather than a concise hand proof.

Proof For Ten Points

Setup

Let L be the triangular lattice generated by (2,0) and (1,√3). Place a unit disc at every lattice point. Adjacent discs are tangent, so the packing has no overlapping interiors. Boundary choices do not change any area computation below. A fundamental parallelogram has area 2√3. In the quotient by the lattice, there is one disc center and covered area pi per fundamental parallelogram, up to measure-zero boundary identifications. Therefore the covered density is

rho = pi / (2 sqrt(3)) = pi / sqrt(12) = 0.906899... > 0.9.

Random translation

Translate the entire infinite packing by a vector chosen uniformly from one fundamental parallelogram. Translations are understood modulo the lattice. For any fixed point p, the set of translations that cover p has the same area fraction as the covered part of the quotient cell, so the probability that p is covered is exactly rho. Boundary cases do not affect this probability because the set of translations putting p on a disc boundary has area zero.

Expectation

Let P be any set of 10 points, and let U be the number of points of P not covered by the random translate. By linearity of expectation,

E[U] = 10(1 - rho) = 10(1 - pi / sqrt(12)) = 0.9310... < 1.

The random variable U is a nonnegative integer. If every translate missed at least one of the 10 points, then U ≥ 1 for every translate, forcing E[U] ≥ 1. That contradicts the inequality above. Hence some translate has U = 0, so it covers all 10 points.

Only finitely many discs from that infinite packing are needed: for each of the 10 points, choose one disc that covers it, then delete duplicate choices. The resulting subcollection has at most 10 discs, is still pairwise non-overlapping, and covers the given point set.

How The Minimum Would Be Determined

Define C(n) to mean "every set of n plane points can be covered by non-overlapping unit discs." Define k = min { n : C(n) is false }, if such an n exists.

The property C(n) is downward monotone: if every n point set is coverable, then every smaller point set is also coverable. If sigma_2 denotes the largest n for which C(n) holds, then k = sigma_2 + 1.

To prove the exact value of k, two independent pieces are needed:

  1. A lower-bound theorem: prove C(k - 1), meaning every point set of size k - 1 is coverable.
  2. An upper-bound certificate: give one explicit k-point configuration and prove that no non-overlapping unit-disc family can cover it.

The 10-point proof above gives the lower bound k ≥ 11. Later work improves this to k ≥ 13, so every 12-point set is coverable. In the other direction, Aloupis, Hearn, Iwasawa, and Uehara give a hand-checkable 50-point noncoverable configuration, so k ≤ 50. They also report a 45-point configuration found by exhaustive computer search. In sigma_2 notation, the reported computer-assisted status is 12 ≤ sigma_2 ≤ 44, equivalently 13 ≤ k ≤ 45.

Computation can help search for counterexamples and certify finite case splits, but it does not replace the two proof obligations above. A numerical search becomes a proof only after every geometric case is bounded with exact inequalities or another checkable certificate.

In principle, these fixed-size claims can be expressed in the first-order theory of the reals. For the universal lower bound C(n), put universal quantifiers over the n point coordinates and ask for a finite disjunction over m ≤ n disk centers with pairwise center distances at least 2 and every point within distance 1 of some center. For a proposed counterexample, fix the point coordinates and prove the negation of that existential condition. Exact real-algebraic decision methods can decide such sentences, but they are usually far too large to be useful directly. A practical mathematical certificate is therefore a finite, checkable reduction using exact inequalities, interval bounds, or verified case splits.
