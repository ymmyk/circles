Unit Disc Covering Proof

Answer

Yes. Given any 10 points in the plane, some finite subcollection of a translated hexagonal packing of unit discs covers all 10 points. The discs in that packing are pairwise disjoint except for allowed tangencies. If an alternate rule forbade even tangency, replace the lattice spacing 2 by 2 + epsilon for sufficiently small positive epsilon. The density remains greater than 0.9, so the same expectation proof gives strictly separated closed unit discs. The app and puzzle comments allow tangency, so the tangent hexagonal packing is the intended model here.

The second question has a different answer than a search for a 10-point counterexample suggests: 10 is not a failure case. For the general problem, the exact minimum number of points that can force failure is an open discrete-geometry problem. Let that minimum be k. Current published bounds place it in the range 13 ≤ k ≤ 45. The upper end is computer-assisted in the cited 2012 work; the same literature also gives a hand-checkable analytic noncoverable configuration with 50 points.

Proof For Ten Points

Setup

Let L be the triangular lattice generated by (2,0) and (1,√3). Place a unit disc at every lattice point. Adjacent discs are tangent, so the packing has no overlapping interiors. Boundary choices do not change any area computation below. A fundamental parallelogram has area 2√3, and it contains one unit disc worth of area. Therefore the covered density is

rho = pi / (2 sqrt(3)) = pi / sqrt(12) = 0.906899... > 0.9.

Random translation

Translate the entire infinite packing by a vector chosen uniformly from one fundamental parallelogram. For any fixed point p, the probability that p is covered is exactly the density rho. Boundary cases do not affect this probability because the set of translations putting p on a disc boundary has area zero.

Expectation

Let P be any set of 10 points, and let U be the number of points of P not covered by the random translate. By linearity of expectation,

E[U] = 10(1 - rho) = 10(1 - pi / sqrt(12)) = 0.9310... < 1.

The random variable U is a nonnegative integer. If every translate missed at least one of the 10 points, then U ≥ 1 for every translate, forcing E[U] ≥ 1. That contradicts the inequality above. Hence some translate has U = 0, so it covers all 10 points.

Only finitely many discs from that infinite packing are needed: discard every disc that contains none of the 10 points. The remaining discs are still pairwise disjoint and cover the given point set.

How The Minimum Would Be Determined

Define C(n) to mean "every set of n plane points can be covered by disjoint unit discs." Define k = min { n : C(n) is false }, if such an n exists.

To prove the exact value of k, two independent pieces are needed:

  1. A lower-bound theorem: prove C(k - 1), meaning every point set of size k - 1 is coverable.
  2. An upper-bound certificate: give one explicit k-point configuration and prove that no disjoint-unit-disc family can cover it.

The 10-point proof above gives the lower bound k ≥ 11. Later work improves this to k ≥ 13, so every 12-point set is coverable. In the other direction, dense point configurations give finite upper bounds; current literature states the equivalent bound 12 ≤ sigma_2 ≤ 44, where sigma_2 is the largest n for which C(n) holds. Equivalently, 13 ≤ k ≤ 45. A conservative fully analytic upper bound from the 2012 paper is k ≤ 50.

Computation can help search for counterexamples and certify finite case splits, but it does not replace the two proof obligations above. A numerical search becomes a proof only after every geometric case is bounded with exact inequalities or another checkable certificate.
